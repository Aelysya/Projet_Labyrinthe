Fonctionnement:

Une fois le programme lancé, la console demandera une entrée utilisateur correspondant au fichier .txt contenant le labyrinthe à résoudre.
Lorsqu'un nom de fichier correct est donné, le programme s'occupe d'afficher les différentes résolutions du labyrinthe donné pour chacune de nos heuristiques.
Si le nom du ficher est incorrect, le programme redemande le nom du fichier à lire. Si la grille de labyrinthe contenue dans le fichier lu est incorrecte, le programme s'arrête.
